Fixed bar added front-end bar to your site. You can customize it (color, position, size) and added to bar any widgets.

== Description ==

Also you can specify page IDs, where fixed bar to be on|off state.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Plugin Name screen to configure the plugin
